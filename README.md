# GTAWantedDisplayTestPlugin

GTAWantedDisplayMod のサーバー側テスト用プラグインです。

## 動作条件

- Minecraft 1.15.2
- PaperMC 1.15.2

## API
[ここ](docs/API.md)を参照してください。

### JavaDoc
[ここ](https://teamkun.github.io/GTAWantedDisplayStarPlugin/)を御覧ください。

## 仕様

### コマンド
+ /setstar
#### つかいかた
+ /setstar <now> <max> <blinking>
+ now: 現在の(変更した)手配度
+ max: 最高手配度
+ blinking: 点滅

### 特筆事項

プレイヤーが(Modを)導入してなくても使用可能です
